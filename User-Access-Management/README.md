# User & Access Management

This section covers Microsoft 365 user lifecycle tasks including account creation, license assignment, password resets, blocking/unblocking sign-in, and user restoration.

### Scenarios Practiced
- Create users (standard and admin)
- Assign and remove licenses
- Reset passwords and force password change
- Block and unblock user sign-in
- Delete and restore users
